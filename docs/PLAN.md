# 🐱 猜數字遊戲 - 實作計畫

## 🎨 設計風格

| 項目 | 選擇 |
|-----|------|
| 角色 | 🐱 小貓咪（會做表情變化） |
| 背景 | 粉色點點 (Polka Dot) |
| 按鈕 | 圓圓胖胖的糖果風格 |
| 主題色 | 粉色系（粉紅 + 淡粉 + 奶油白） |

## 🐱 小貓咪表情設計

| 遊戲狀態 | 小貓咪做什麼 |
|---------|-------------|
| 等待猜測 | 懶洋洋打哈欠 😴 |
| 數字太小 | 往上指 + 皺眉 👆😾 |
| 數字太大 | 往下指 + 搖頭 👇😿 |
| 猜對了！ | 開心跳躍 + 撒星星 🎉✨ |
| 失敗（範圍沒了） | 哭哭 😿 |

## 📋 實作步驟（Vue 3 CDN 方式）

### 第一步：建立一個 HTML 檔案
建立 `index.html`，放基本框架
- 告訴瀏覽器這是 HTML5 網頁
- 加入 Vue 3 CDN 連結
- 加入 Google Fonts 圓潤字型

### 第二步：做點點背景
用 CSS 畫出粉色背景 + 白點點

### 第三步：建立「遊戲卡片」
一個白色/淡粉色的框框放在中間
- 放小貓咪的「家」
- 放輸入框和按鈕
- 放訊息顯示區

### 第四步：畫一隻 CSS 小貓咪
用 CSS 畫出圓圓的貓咪臉（不用圖片！）
- 圓形當臉、圓形當耳朵
- 兩個圓點當眼睛
- 弧線當嘴巴
- 用 `v-if` 切換不同表情

### 第五步：建立「盒子」存放遊戲資料
用 Vue 的盒子功能，存放：
- `答案` - 電腦選的數字（0~100）
- `目前範圍` - 最小值(0)和最大值(100)
- `猜測次數` - 記錄猜了幾次
- `遊戲狀態` - 還在玩、贏了、輸了

### 第六步：實作「猜數字」功能
1. 按下按鈕時檢查輸入的數字
2. 比較數字：比答案大？還是比答案小？
3. 根據結果縮小範圍
4. 更換小貓咪表情
5. 次數 +1

### 第七步：美化與動畫
- 按鈕 hover 變大、變亮
- 小貓咪跳躍動畫
- 勝利時撒星星效果
- 輸入框 focus 時發光

### 第八步：加入「再玩一次」功能
- 按下後重置所有資料
- 小貓咪回到初始表情
- 範圍回到 0~100

## 🎮 遊戲玩法

1. 電腦心裡想一個數字（0 到 100）
2. 你猜一個數字
3. 電腦說「太小了 👆」或「太大了 👇」
4. 範圍變小，你再猜
5. 一直猜到對為止 🎉

## 💡 遊戲技巧：二分搜尋法

每次猜**中間的數字**，可以最快縮小範圍：
- 100 個數字只需要 **7 次** 就能找到答案！
