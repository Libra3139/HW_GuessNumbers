# GitHub 偵察機 - 開發計畫

## 🖼️ 當你在網頁上打字、然後看到照片時，發生了什麼事？

想象你要寄信給聖誕老公公拿禮物：

### 第一關：寫信（輸入資料）
你在輸入框打字，就像在信紙上寫下「我想找小明」。瀏覽器把這些文字收進一個小袋子裡。

### 第二關：寄信（按下按鈕）
你按下「搜尋」鈕，就像把信投進郵筒。瀏覽器把小袋子綁上，準備寄出去。

### 第三關：郵差送的信（網路傳送）
網路像一條看不見的高速公路，把小袋子送到 GitHub 伺服器那邊。

### 第四關：聖誕老公公找禮物（伺服器處理）
GitHub 伺服器收到信後，去大大大大的資料庫裡找「小明」這個人，找到他的大頭貼照片。

### 第五關：收到禮物（回傳資料）
伺服器把找到的照片放進小袋子，透過網路高速公路傳回來。

### 第六關：打開禮物（顯示結果）
瀏覽器收到照片後，像魔術一樣把照片變到網頁上，你就看到啦！

## 📝 開發步驟

### 第一天：建立網頁骨架
1. 用 Vite 建立 Vue 3 專案
2. 寫出一個輸入框 + 一個按鈕 + 一個放照片的位置

### 第二天：實作搜尋功能
1. 當按鈕被按下時，去 GitHub API 找人
2. 找到人之後，把大頭貼網址拿回來

### 第三天：美化網頁
1. 幫照片加上圓圓的邊框
2. 讓 loading 的時候顯示「正在找人中...」
3. 如果找不到人，顯示「找不到這個人QQ」
